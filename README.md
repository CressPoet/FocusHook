
# FocusHook - 随机专注提示音程序 / Random Interval Focus Timer

## 中文

### 简介
**FocusHook** 是一个基于 Python Tkinter 开发的轻量级桌面小工具。它通过在设定的时间范围内随机触发提示音，帮助使用者将飘散的注意力重新拉回当前的事务上。非常适合需要长时间面对电脑工作、学习或进行正念训练的场景。

**原理**：不同于固定时间，通常为随机间隔3-5分钟的番茄钟，不可预测的随机铃声能形成一种“注意力钩子”，强制大脑定期进行自我觉察，检查当前是否处于分心状态。

### 功能特点
*   **自定义随机区间**：可自由设定随机提示音的最小与最大时间间隔（分钟）。
*   **实时倒计时**：界面直观显示当前周期的剩余时间。
*   **多线程运行**：倒计时与核心逻辑在后台线程运行，UI 界面流畅不卡顿。
*   **原生提示音**：采用 Windows 系统底层 API 响铃，无需依赖外部音频文件或复杂的音频库。

### 项目结构
```plaintext
tishi/
├── build/          # PyInstaller 编译生成的临时缓存目录
├── dist/           # 编译产物目录
│   └── main.exe    # 打包好的 Windows 可执行程序
├── 1.py            # 测试/备份脚本
├── main.py         # 程序主代码（GUI 与业务逻辑）
└── main.spec       # PyInstaller 配置文件
```

### 运行环境与依赖
*   **操作系统**：Windows (由于使用了 `winsound` 库，目前仅支持 Windows 系统)
*   **Python 版本**：Python 3.x
*   **内置库**：`tkinter`, `threading`, `random`, `time`, `winsound` （无需额外 `pip install`）

### 使用方法
1.  克隆或下载本仓库。
2.  运行主程序：
    ```bash
    python main.py
    ```
3.  在界面中输入你期望的最小间隔和最大间隔（例如：3 到 5 分钟）。
4.  点击 **开始** 按钮，程序将进入后台随机倒计时；点击 **停止** 可随时结束。
5.  如果你不想配置 Python 环境，可以直接运行 `dist/main.exe`。

## English

### Introduction
**FocusHook** is a lightweight desktop utility developed in Python using Tkinter. It generates alert beeps at random intervals within a user-defined time range to help you snap out of distractions and pull your wandering mind back to the task at hand. It is ideal for long study sessions, deep work, or mindfulness check-ins.

**Why it works**: Unlike a standard Pomodoro timer with predictable intervals, unpredictable random beeps act as an "attention hook," prompting your brain to perform regular self-checks on whether you are currently distracted.

### Features
*   **Customizable Random Ranges**: Flexible configuration of minimum and maximum intervals (in minutes).
*   **Real-time Countdown**: Clear visual display of the remaining time for the current cycle.
*   **Multi-threaded Architecture**: The timer logic runs seamlessly in a background thread, keeping the GUI responsive.
*   **Native Beep Alerts**: Uses the low-level Windows API (`winsound`) to trigger audio alerts without requiring external audio assets or heavy third-party libraries.

*   **Python Version**: Python 3.x
*   **Built-in Libraries**: `tkinter`, `threading`, `random`, `time`, `winsound` (No external dependencies required)

### How to Use
1.  Clone or download this repository.
2.  Run the main script:
    ```bash
    python main.py
    ```
3.  Input your preferred Minimum Interval and Maximum Interval (e.g., 3 to 5 minutes).
